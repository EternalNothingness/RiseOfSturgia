# Customized Sturgia

This mod replaces the 2-hand maces of the (heroic) line breakers with the original 2-hand axes.

## Installation

Put this mod in the Modules directory of your Bannerlord installation. If you use the steam version on Windows, this can usually be found at the path 'C:\Program Files (x86)\Steam\steamapps\common\Mount & Blade II Bannerlord'. Otherwise follow some online tutorial that shows you how to browse local files of an installed game.

## Editing this mod

Follow this [steam guide](https://steamcommunity.com/sharedfiles/filedetails/?id=2053822450), but edit the files in this mod, not the files of your main installation!
